pdsutil package
===============

Introduction
============

This package has two major areas of functionality:

* Datasets

  * load data sets from

    * sql
    * list of lists
    * list of dictionaries
    * csv

  * render to

    * excel
    * csv
    * json
    * latex

* SQL

  * DDL runner

  * DML runner

* DbUtil

  * Connection Helper
  * Cursor Helper

    * Automatically converts bind statements
    * Handles returning on insert in a consistent manner for sqlite and postgress

  * Cursors
    Issues cursors and closes them without using a contex
